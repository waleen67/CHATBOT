Simple Python Chatbot
A simple rule-based chatbot built using Python.
This project demonstrates how to create a basic text-based chatbot that interacts with users through the terminal and responds to predefined commands

Project Overview
This chatbot uses a dictionary to map user inputs to predefined responses.
For each recognized input, the bot selects a response randomly to make conversations feel more natural.
The project is mainly intended for beginners who want to practice core Python programming concepts.

Features
Rule-based chatbot
Terminal interaction
Random responses for the same input
Simple and clean code structure
Beginner-friendly project
Technologies Used
Python 3
Standard Python libraries only
How to Run the Project
Make sure Python is installed on your system:
python --version
